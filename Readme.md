# REST API Endpoints

## Authentication

- POST /api/auth/login
- POST /api/auth/logout
- POST /api/auth/register
- GET /api/auth/user

## Users

- GET /api/users
- GET /api/users/:id
- POST /api/users
- PUT /api/users/:id
- DELETE /api/users/:id

## Products

- GET /api/products
- GET /api/products/:id
- POST /api/products
- PUT /api/products/:id
- DELETE /api/products/:id
